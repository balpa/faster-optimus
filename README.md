# Faster Optimus Chrome Extension

## Kurulum

1. Chrome tarayıcınızda `chrome://extensions/` adresine gidin
2. Sağ üst köşeden "Geliştirici modu"nu aktif edin
3. "Paketlenmemiş öğe yükle" butonuna tıklayın
4. Bu klasörü seçin

## Geliştirme

Değişiklik yaptıktan sonra `chrome://extensions/` sayfasında extension'un altındaki yenile butonuna tıklayın.
